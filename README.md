# Terraform Practice Repository

This repository contains early Terraform configurations created while learning AWS fundamentals and Infrastructure as Code concepts.

## Purpose
The goal of this repository was to:
• Understand Terraform syntax and workflow  
• Practice provisioning AWS resources  
• Learn state management and resource dependencies  

## Status
This repository represents early experimentation and learning.
More complete, production-style Terraform projects are available in newer repositories.
